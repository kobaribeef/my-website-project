The Flexible Box Layout Model (Flexbox) is a layout model designed for one-dimensional content. It is excellent at taking a group of items which have different sizes, and gives back the best layout for those items. Flexbox can be useful for creating small-scale layouts and is responsive and mobile-friendly.

Before we start using flexbox to alter a group of items. We'll be using the HTML code below which shows a group of items in a div element for visualization.

In order to use flexbox, you need to delcare that you are using the flex formatting context and not regular block and inline layout. To use flexbox, change the value of the display property to flex. 

            
                .container {
                    display: flex;
                    }

The group of items in the div element with class "container" will start showing some flexbox behavior and display the items as a row.

If you wanted to change the direction of your items to display as a column, you could use the flex-direction property. Items are displayed as a row intially because the initial value of flex-direction is a row. 

            
                .container {
                    display: flex;
                    flex-direction: column;
                    }

The group of items will now be displayed as a column since the flex-direction is set to column.

You can also wrap flex items using the flex-wrap property. The initial value of flex-wrap is set to nowrap. This means that if there is not enough space in the contrainer, the items will overflow. 

            
                .container {
                    display: flex;
                    flex-direction: column;
                    flex-wrap: wrap;
                    }

The group of items can now wrap on multiple lines.
