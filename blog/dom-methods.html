DOM methods are methods that can be used to perfom actions on objects or in other words elements of DOM. We use Javascript to change and manipulate HTML elements of a page. To even start changing or manipulating elements, we must first get those elements by using DOM methods.

First, I'll introduce to you the document.getElementById() method. This method allows you to access an HTML element by their id. All you have to do is pass in the id of the element as a string parameter in the method.
This method returns an element object representing the element whose id property matches the specified string. Note that this method returns the first element that matches the id. In the case below, we are getting the element whose id is set to "blog". 

            
                document.getElementById("blog");

The next method I'll introduce is similiar to the first method but gets an element in a different way. This method is called document.querySelector().
With this method, you can get an element based on a CSS selector instead. What this means is that you can get elements by their id, class, or other types of selectors. This method is more flexible and not limited unlike the previous method.
This method returns an element object whos CSS selector also matches the specified string. So you must pass in the parameter as a string but the syntax is different. Here's an example below: 

            
                document.querySelector("#blog");

This method also gets an element whose id matches blog but notice the number sign before blog. This number sign represents an id of an element. Remember this method can get other types of selectors so you must know the syntax in order to grab other selectors.
Here is another example below showing how to get an element by their class attribute instead of id attribute: 

            
                document.querySelector("p.example");

As you can see, the class attribute of the element is "example" but there is a "p" with a dot before the class attribute. This lets the method know you are looking for a element with a class attribute set to "example".
The last method I'll introduce is the document.querySelectorAll(). This method is the same as querySelector but the difference is that this method returns an array of element objects that matches the CSS selector instead of the first element that matches the selector.
Use this method when you want to return multiple element objects that match a certain selector. Here is an example of a method that returns an array of element objects that have their class attribute set to "example": 

                
                    document.querySelectorAll("p.example");
